# Python Tutor

_[Python Tutor](http://pythontutor.com/) (created by Philip Guo) helps people overcome a fundamental barrier to learning programming: understanding what happens as the computer runs each line of code.
Write code in your web browser, see it visualized step by step, and get live help from volunteers.
Related services: [Java Tutor](http://pythontutor.com/java.html),  [C Tutor](http://pythontutor.com/c.html),  [C++ Tutor](http://pythontutor.com/cpp.html),  JavaScript Tutor,  [Ruby Tutor](http://pythontutor.com/ruby.html)_
